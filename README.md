## Course Overview

In this course, we explore the foundations and practical techniques of data visualization using Python.  
The notebooks cover:

- Understanding the role of data visualization in data analysis  
- Working with Pandas for data manipulation  
- Using Matplotlib and Seaborn for static plots  
- Creating advanced and interactive charts with Plotly  
- Best practices for designing readable and meaningful visualizations
